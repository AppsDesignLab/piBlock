To get things running:

1. Make sure you have python installed with the correct $PATH
2. Make sure you have PIP installed
3. Its handy to have PYENV installed
4. Check Pyenv versions
5. Install Kivy
6. Check Python version in Kivy
7. For PiBlock, Install URLLIB3, Twisted, PyCrypto, PyAsn1 with sudo kivy -m pip install <Modulename>
